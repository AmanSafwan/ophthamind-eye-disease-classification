# Trained model weights

Place your exported Keras models in this directory before running the AI service:

```
ai/models/
├── cnn/cnn_final.keras
├── vgg16/vgg16_final.keras
└── resnet50/resnet50_final.keras
```

These files are **not committed to Git** (large binary assets).  
Include them when deploying locally or attach them separately for FYP submission / demo.
